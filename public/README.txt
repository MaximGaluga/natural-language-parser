Как рабоать с программой:

	1. Слова на русском языке
	2. Повторная загрузка файла с уже разобранными словами по морфологическим правилам приведет к дублированию данных
	3. Для того, чтобы добавить разбор новых слов в уже обработанный файл нужно:
		* Создать новый документ
		* Загрузить его на платформу
		* Получить обработанный файл
		* Смержить данные вручную из предыдущего обработанного файла и текущего
